# Seoul Dimming Recommender - Frontend

꼭 읽고 지시대로 이행해주세요!  

## 실행 방법(Startup Instructions)
1. 터미널에서 프론트 폴더로 이동: `cd frontend`
2. 의존성 설치(필수): `npm ci`
   - 만약 `package-lock.json`이 없거나 오류가 나면: `npm install`
3. 개발 서버 실행: `npm run dev`
4. 브라우저에서 접속: http://localhost:5173

> **참고**: 현재는 Mock Data로 동작합니다. 개발 편의를 위해 `RecoService`가 시뮬레이션 데이터를 반환하도록 설정되어 있습니다.
